# GlicoTracker

## 1. Introdução
O GlicoTracker é uma aplicação web focada no controle glicêmico pessoal, desenvolvida para substituir métodos manuais (planilhas) por uma solução tecnológica intuitiva, segura e com foco na privacidade do paciente.

## 2. Tecnologias Utilizadas
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla).
- **Arquitetura**: PWA (Progressive Web App), permitindo instalação local e funcionamento offline.
- **Armazenamento**: LocalStorage do navegador (garantindo soberania e privacidade dos dados ao usuário).
- **Gráficos**: Chart.js para visualização da evolução dos índices.

## 3. Funcionalidades
- Registro de índices glicêmicos por período (Manhã, Tarde, Noite).
- Histórico filtrável por data e período.
- Visualização de médias diárias.
- Gráficos interativos com faixa de segurança personalizada.
- Exportação de relatórios em formato CSV.

## 4. Privacidade e Segurança
Diferente de soluções baseadas em nuvem, o GlicoTracker armazena todos os dados de saúde localmente no dispositivo do usuário, eliminando riscos de vazamento de dados sensíveis e garantindo total conformidade com os princípios de privacidade.
